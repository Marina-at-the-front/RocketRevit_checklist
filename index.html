Настройка синхронизации

Для синхронизации прогресса между всеми участниками команды необходимо настроить Firebase.

Шаг 1: Создайте проект Firebase

  1. Перейдите на Firebase Console
  2. Создайте новый проект или выберите существующий
  3. Перейдите в раздел "Realtime Database"
  4. Создайте базу данных в тестовом режиме
  5. Скопируйте конфигурацию из Project Settings → General → Your apps → Web app

Шаг 2: Настройте правила безопасности

В разделе Realtime Database → Rules установите:

{
  "rules": {
    "rocketrevit-checklist": {
      ".read": true,
      ".write": true
    }
  }
}
